# Stalker G.A.M.M.A Website

Authorized [Stalker G.A.M.M.A](https://discord.gg/stalker-gamma) website




## Run Locally

Clone the project

```bash
  git clone https://github.com/Asterki/gamma-website
```

Go to the project directory

```bash
  cd gamma-website
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
